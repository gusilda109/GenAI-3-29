# Letter Generator

Программа генерирует письмо **официального** или **дружеского** типа объёмом **100–150 слов**, добавляет тему и подпись и сохраняет результат в файл `.txt`.  
Для генерации текста используется отдельный модуль с кодом одногруппника, подключаемый через импорт.

---

## Использование

```bash
python3 main.py --type оф --topic "Запрос справки" --sign "С уважением, Илья" -o letter.txt
